# EyeTrackVR
{: .no_toc }

Open source and *affordable* VR eye tracker platform for [VRChat](https://hello.vrchat.com/) via `OSC` and `UDP` protocol.

This project is in development and is not fully finished
{: .fs-5 .fw-300 }
